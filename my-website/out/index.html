Witch Turn
Witch turn is a D&D initiative management app for GMs, and an ongoing project in my personal time. It uses a room system inspired by Jackbox, and I'm still adding tools to it. All the art is my own work, and it's a fun outlet to be creative. The client side is built on React. The service which manages the rooms and message communication is built on express.js, using the socket.io library to facilitate message passing. The database RESTful API is built on express.js
